Algorithm Complexity

Time Complexity: O(n log n) - where n is the number of unique characters in the input text. This is because:

  • Building frequency table: O(n)
  • Building priority queue: O(n)
  • Building Huffman tree: O(n log n)
  • Generating codes: O(n)

Space Complexity: O(n) - We need to store:

  • Frequency table: O(n)
  • Huffman tree: O(n)
  • Code table: O(n)

Why Huffman Compression? Huffman coding assigns variable-length codes to characters based on their frequencies. More frequent characters get shorter codes, resulting in overall compression.

Image Compression Details

The image is compressed using a simplified version of JPEG compression techniques:

  • Color space conversion (RGB to YCbCr)
  • Downsampling of chrominance components
  • Discrete Cosine Transform (DCT) on 8x8 blocks
  • Quantization to reduce high-frequency components
  • Huffman coding for final compression

Note: This is a demonstration of lossy compression. Some image quality is sacrificed to achieve higher compression ratios.
